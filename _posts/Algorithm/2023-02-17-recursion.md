# 재귀  
## 알고리즘 설명  
* 재귀: 하나의 함수에서 자기 자신을 다시 호출해 작업을 수행하는 알고리즘  
* 재귀 함수의 조건: 특정 입력에 대해서는 자기 자신을 호출하지 않고 종료되어야 함(Base condition)  
* 모든 입력은 base condition으로 수렴해야 함  
* 재귀함수가 자기 자신을 부를 때 스택 영역에 계속 누적이 됨  
## Tip  
* 함수를 정의!  
* Base condition 잡기!  
* 귀납적으로 생각하기!
